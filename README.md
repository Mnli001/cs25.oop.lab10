## Зорилго
Энэхүү лабораторийн ажлаар Java хэлний **Женерик (Generics)**, **Дотоод Класс (Inner Classes)**, болон **Ламбда илэрхийлэл (Lambda expressions)**, **Stream API** зэрэг ойлголтуудыг практик дээр хэрэгжүүлж сурах зорилготой.

## Гүйцэтгэсэн дасгалууд
Энэхүү репозиторид дараах 4 дасгалын шийдлийг оруулсан болно:

1. **Дасгал 1: Женерик ба Wildcards (`Zoo.java`, `ZooApp.java`)**
   - Дээд болон доод хязгаарлалттай (`? extends T`, `? super T`) женерик метод бичиж, объектуудыг аюулгүй шилжүүлэх үйлдэл хийсэн.
2. **Дасгал 2: Дотоод Класс ба Итераци (`Library.java`, `LibApp.java`)**
   - `Iterable` интерфэйсийг хэрэгжүүлсэн дотоод класс үүсгэж, өгөгдсөн оноос хойшхи номуудыг шүүж буцаадаг итератор бичсэн.
3. **Дасгал 3: Ламбда ба Функциональ (`StringProcessor.java`)**
   - Java-ийн `Predicate`, `Function`, `Consumer` интерфэйсүүдийг ашиглан тэмдэгт мөрийг шүүх, хувиргах, хэвлэх үйлдлүүдийг хийсэн.
4. **Дасгал 4: Урсгал ба Захиалгат Эрэмбэлэлт, Нэгтгэл (`ProductAnalyzer.java`)**
   - `Stream API` ашиглан бүтээгдэхүүнүүдийг үнээр нь шүүж, эрэмбэлж, мөн категориор нь бүлэглэж (`groupingBy`) харуулсан.

## Хэрхэн ажиллуулах вэ?
Дасгал бүрийн үндсэн класс (Main method агуулсан файл) дээр баруун товчоо дараад **Run Java** (VS Code, IntelliJ, Eclipse зэрэг IDE дээр) сонгож ажиллуулна.
